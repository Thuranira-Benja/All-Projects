🛠️ Step-by-Step: Set Up Data in Django Admin

Go to your admin panel and follow the steps below:

1️⃣ Add Classes (SchoolClass)

Go to: School classes > Add

Create entries like:

  • Form 1
  • Form 2
  • Form 3
  • Form 4

You must create at least 1 class.

2️⃣ Add Subjects (Subject)

Go to: Subjects > Add

Add subjects like:

  • Mathematics
  • English
  • Biology
  • Chemistry
  • History

Each subject name must be unique and correctly spelled.

3️⃣ Add Teachers (Teacher)

Go to: Teachers > Add

Fill in details like:

  • Name: e.g., Mr. Otieno
  • Phone number: e.g., +254712345678
  • Max lessons per day: e.g., 4
  • Max lessons per week: e.g., 20
  • Subjects: Select multiple (e.g., Math, Chemistry)
  • Preferred classes: Select the Forms they teach (e.g., Form 1, Form 2)
✅ Ensure:
- Each subject has at least one teacher
- Each teacher has enough capacity (lessons/day & week)

4️⃣ Allocate Subjects to Classes (SubjectAllocation)

Go to: Subject allocations > Add

Example entry:

  • School class: Form 1
  • Subject: Mathematics
  • Lessons per week: 5
  • Requires double period: ✔️ if needed

🔁 Repeat for each subject per class. Sample:

ClassSubjectLessonsDouble Period
Form 1Math5❌
Form 1English4✅
Form 2Chemistry3✅

Ensure each allocation matches with a teacher who teaches that subject and prefers that class.

5️⃣ Add Time Slots (TimeSlot)

Go to: Time slots > Add

Create enough periods per day (e.g., 6 per day, Monday to Friday).

Example entries:

DayStart TimeEnd TimeIs Break?
Monday08:0008:40❌
Monday08:4009:20❌
Monday09:2010:00❌
Monday10:0010:20✅

🧠 Tip: If you need 30 lessons per week, make sure your time slots can fit that.

✅ After Admin Setup

Go to the Dashboard and click Generate Timetable.

You will be redirected back to the dashboard with a success message if everything is correct.
